# CloudSeed [JUCE]

Port of CloudSeed VST plugin (https://github.com/ValdemarOrn/CloudSeed) with new GUI based on JUCE Framework

---
#### Features:<br>
[x] multi channel support<br>
[x] macOS version<br>

2. mkdir build && cd build && cmake .. && make
